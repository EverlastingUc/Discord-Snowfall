## Overview

Discord Snowfall adds a gentle, falling snow effect to your Discord client. The snowflakes fall naturally with varying sizes and speeds, creating a cozy winter atmosphere.

---

## Features

- Realistic snowfall effect.
- Random flake sizes and speeds.
- Smooth animation.
- Low performance impact.
- No configuration needed.

---

## Installation

1. Install Tampermonkey or Violentmonkey
2. Click [here](https://raw.githubusercontent.com/EverlastingUc/Discord-Snowfall/main/snowfall.user.js) to install
3. Open Discord
4. Snow will start falling automatically

---

## How To Use

Simply install the script and open Discord. Snow will begin falling immediately. No extra buttons or setup required.
